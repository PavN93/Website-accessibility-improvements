# Horiseon - website improvements
## About Horiseon
Currently the best practices in subject of web accessibility requires the developers to follow the important set of rules. This is what Horiseon is specialized in. In order to achieve the company's goal the improvements takes place in three main areas:
* Search Engine Optimization
* Online Reputation Management
* Social Media Marketing
## Code refactoring
As an example the presented website has been refactored to increase its accessibility in the following steps:
* Semantic HTML elements added in place of the universal tags to divide the website into more meaningful way
* CSS elements is now consolidated to avoid repetitions and decrease complexity
* Added website title is short and descriptive
* Addition of alt attributes to all images
